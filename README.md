# EuroMalin 💰

**Cashback, économies et budget malin.**

EuroMalin est un site clair et utile pour économiser au quotidien : comparatifs cashback, astuces budget, bons plans et revenus complémentaires.

## Structure

```
/
├── index.html           ← Accueil
├── articles.html        ← Tous les articles (+ recherche)
├── cashback.html        ← Catégorie Cashback
├── economies.html       ← Catégorie Économies
├── budget.html          ← Catégorie Budget
├── revenus.html         ← Catégorie Revenus
├── 404.html             ← Page erreur
├── assets/
│   ├── style.css        ← Design system
│   └── script.js        ← Recherche en direct
└── articles/            ← 21 articles HTML
```
