Politique de confidentialité

1. Qui sommes-nous ?

EuroMalin est un site de cashback, bons plans et économies. Adresse : euromalin.com.

Contact : via Telegram uniquement.

2. Quelles données sont collectées ?

Ce site utilise Google Analytics (via le tag G-DXH0N60DDB) pour mesurer l'audience :

  • Pages visitées, durée de visite, provenance (Google, réseau social, lien direct)
  • Type d'appareil, navigateur, pays approximatif
  • Adresse IP (anonymisée par Google)

Aucune donnée personnelle (nom, email, adresse) n'est collectée via ce site. Les emails sont uniquement collectés via la newsletter (inscription volontaire).

3. Cookies

Ce site utilise :

  • Google Analytics (_ga, _gid, _gat) — cookies de mesure d'audience, conservés 14 mois max.
  • Cookie consent (Osano) — cookie de préférence pour mémoriser ton choix (acceptation/refus).

Aucun cookie publicitaire, aucun cookie tiers autre que Google Analytics.

4. Google Analytics

Ce site utilise Google Analytics 4 (propriété G-DXH0N60DDB). Les données sont anonymisées. Tu peux :

  • Refuser le tracking via la bannière de cookies
  • Installer le module complémentaire de désactivation GA
  • Consulter la politique de Google : policies.google.com/privacy

5. Google Tag Manager

Google Tag Manager n'est pas encore utilisé sur ce site. Actuellement, le tag Google Analytics est installé directement dans le code HTML. Si un GTM est ajouté plus tard, cette page sera mise à jour.
